# Helm Charts

Add this repo:

```sh
helm repo add strikerlulu https://strikerlulu.github.io/helm-charts
helm repo update
```

Available charts:
- `homepage`

Search:

```sh
helm search repo strikerlulu
```

Chart docs live under `charts/<name>`. This repo is published as a Helm repository on GitHub Pages with index generated by the chart-releaser action.
